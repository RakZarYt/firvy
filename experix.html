Полный список ExperiX 0.4:

  • ТЕГИ (ВКЛ.)
    • [DirtBlocks] - земля
    • [LogBlocks] - бревно
    • [LeaveBlocks] - листва
  • ТАБЫ ДЛЯ КРЕАТИВ
    • [Блоки] - блоки, без блоков из других миров
    • [ПРЕДМЕТЫ] - предметы, без предметов из других миров
    • [ДРУГОЕ] - другие предметы, без тех что используются в других мирах
    • [Милоранта] - блоки, предметы и т.п. из мира Милоранты
  • РУБИНЫ
    • [Рубин] - предмет, похожий на алмаз
    • [Рубиновая руда] - руда, для получения рубина
    • [Ачивка за получение рубина] - ачивка даётся при получении рубина
    • [Блок рубина] - блок из рубинов
    • [Крафты]: рубин, блок рубина, рубин #2
  • ЛАГАРИТ
    • [лагарит] - предмет, для создания портала
    • [руда лагарита] - для добычи лагарита
    • [блок лагарита] - для использования лагарита
    • [крафты]: блок лагарита, лагарит, лагарит #2
  • БЕЛЫЙ АЛМАЗ
    • [Белый алмаз] - это материал из лагарита и рубина, очень прочный и эффективный
    • [Кирка] - кирка из белого алмаза
    • [Броня] - кирка из белого алмаза
    • [Меч] - кирка из белого алмаза
    • [Крафты]: белый алмаз, шлем, нагрудник, штаны, ботинки, кирка, меч
  • БЕЛЫЙ АЛМАЗ
    • камень этого мира более прочный, чем обычный, более тёмный
    • бревно этого мира, может понравится лишь тем, кто ценит...
    • доски из бревна, получаются дорогие, хорошо продаются
    • листва, не пропадает и это баг!
    • структура "дерева" - обычное дерево в этом мире
    • саженец - говорят, из него вырастает это дерево
    • Скрипт для саженец -> говорят, из него вырастает дерево Милоранты
    • биом, типичный в этом мире
    • портал, в измерение Милоранты!
    • Милоран - требуется для открытия портала
    • Частицы - частицы из портала в Милоранту
    • крафты: палки, доски
    • ЗЕМЛЯ МИЛОРАНТЫ
      • Земля в Милоранте
      • Дёрн в Милоранте
      • Скрипт для земля -> дёрн
      • Скрипт для дёрн -> земля
