Добро пожаловать! На данном мини ресурсе-проекте будет содержаться информация связанная с моими личными и нетолько проектами / блоги, новости и различная интересная информация.

Для любой связи со мной используйте телеграм: rakzaryt

Проекты, которые вы можете здесь найти: "DivCook" - модификация для Minecraft Forge 1.16.5 / ForgeOptifine 1.16.5, добавляет много различной еды, с собственными механиками для кулинарии. Над этим модом работают: Roman_Voran, Hell_Artem, частично GiverYT. Мод был создан с использованием MCreator 2022.1, следующие моды тоже использую данную программу: "AloreToolsMod" - модификация для Minecraft Forge 1.16.5 / ForgeOptifine 1.16.5, добавляет много различных предметов и нетолько. Разработчик: GiverYT. "ExperiX" - модификация для Minecraft Forge 1.16.5 / ForgeOptifine 1.16.5, учебная модификация Roman_Voran.
